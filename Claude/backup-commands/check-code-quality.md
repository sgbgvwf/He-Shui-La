# 检查代码质量

全面检查 Unity C# 代码质量，包括编译检查和代码规范验证。

## 执行过程

1. **检查 Unity 编译状态**
   - 打开 Unity 编辑器，确认控制台无编译错误和警告

2. **检查 C# 代码规范**
   - 检查命名规范（PascalCase 公开成员、camelCase 私有成员）
   - 检查是否遵循项目行为准则（无魔法数字、enum 优先、最小作用域等）

3. **运行单元测试**
   - 打开 Unity Test Runner（Window > General > Test Runner）
   - 运行所有测试（EditMode + PlayMode）
   - 确保全部通过

4. **修复发现的问题**
   - 修复所有编译错误
   - 修复所有代码规范违规
   - 确保所有测试通过

## 验证标准

- ✅ Unity 编译无错误
- ✅ 无代码规范违规
- ✅ 单元测试全部通过
